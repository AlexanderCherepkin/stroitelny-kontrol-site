# 🧭 Graphify — шпаргалка «задача → команда»

> Бонус к ролику «Graphify» (Никита Велс | AI). Распечатай или держи открытым.
> У Graphify 40+ команд — вот те, что реально нужны, под человеческие задачи.
> 💰 = тратит токены · 🆓 = бесплатно (локальный AST) · ⚠️ = осторожно

---

## 🟢 СНАЧАЛА: построить граф (один раз)

| Хочу | Команда |
|---|---|
| Построить граф проекта | `graphify .` 🆓 |
| ...но проект маленький (<100 файлов) | **Не строй.** Грэп и так дёшев, граф не сэкономит |
| Освежить граф после правок | `graphify . --update` 🆓 |
| Авто-обновление после каждого коммита | `graphify hook install` 🆓 (безопасный post-commit) |

> В PowerShell пиши `graphify .` **без** слэша спереди (`/` там — путь).

---

## 🔍 ГЛАВНОЕ: спрашивать проект (вместо слепого поиска по файлам)

| Хочу | Команда |
|---|---|
| Понять, как устроена фича | `graphify query "как тут работает авторизация"` 💰 дёшево |
| Найти связь между двумя файлами | `graphify path "AuthService" "LoginPage"` 💰 дёшево |
| Объяснить один класс/функцию | `graphify explain "PaymentService"` 💰 дёшево |

Это и есть смысл Graphify: Claude идёт по карте к нужным файлам, а не перечитывает
половину проекта. Дёшево, потому что читается граф, а не сырой код.

---

## 📚 ЭКСПОРТ (если нужна база знаний)

| Хочу | Команда |
|---|---|
| Вики по проекту (markdown) | `graphify . --wiki` 💰 дёшево |
| Обсидиан-база | `graphify . --obsidian` 💰 дёшево |
| Картинку графа (SVG) | `graphify . --svg` 🆓 |
| Для Gephi/yEd | `graphify . --graphml` 🆓 |

---

## 🛠 КОГДА ГРАФ «КРИВОЙ»

| Симптом | Команда |
|---|---|
| Кластеры слишком крупные | `graphify . --cluster-only --resolution 1.5` 🆓 |
| Утилиты-хабы засоряют картину | `graphify . --cluster-only --exclude-hubs 99` 🆓 |
| «Призрачные» узлы после рефакторинга | `graphify extract . --force` 🆓 |
| Граф огромный (>5000 узлов), HTML тормозит | Не открывай HTML, спрашивай через `graphify query` |

---

## ⚠️ ОПАСНАЯ ЗОНА — три способа выстрелить себе в ногу

| Команда | Чем опасна | Что делать |
|---|---|---|
| `pip install graphify` | Пакет на PyPI — **`graphifyy`** (две `y`). Эта поставит не то | `uv tool install graphifyy` |
| `graphify . --mode deep` | Прогоняет документы через модель = **тратит токены**. На проекте без документации — деньги на ветер | По умолчанию `graphify .` (🆓 AST). `--mode deep` только если много PDF/документов |
| `graphify claude install` | Вшивает PreToolUse-хук. **На свежих сборках Claude Code он слетает** и блокирует чтение файлов | Не ставь без нужды. Нужно авто-обновление — бери безопасный `graphify hook install` (post-commit) |

**Откат, если хук уже сломал Claude:** удали блок `hooks` с `graphify` из
`~/.claude/settings.json` → перезапусти Claude Code. Граф останется.

## 🧠 ПРАВИЛО ПРО ЭКОНОМИЮ (чтобы не разочароваться)

«71× меньше токенов» из обзоров — это **крайний случай для огромных монорепозиториев**, не твой типичный случай:

| Размер проекта | Реальная экономия |
|---|---|
| < 100 файлов | почти ноль — не ставь |
| 100–500 файлов | ≈ 6–15× на запрос |
| > 500 файлов | 30× и выше |

Граф — для больших незнакомых проектов. На маленький проект из 50 файлов он не нужен.
